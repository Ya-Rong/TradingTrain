p 的用法

代表文章段落文字

a 的用法

超連結 href 跳出連結網址覆蓋網頁
超連結 target 自動開新分頁

href 的用法

href 屬性指定 Web 資源的位置，從而定義當前元素或當前文檔與此屬性定義的目標錨點或資源之間的鏈接或關係。

src 的用法

“ src ” 屬性只是將資源嵌入當前文檔中元素定義的位置。

class命名

1.元素避免直接用 相對關係 命名
2.色彩等樣式避免直接用 顏色 命名，改以風格命名
3.Layout 避免用 位置 命名

BEM方式
以「區塊__元素 — 樣式」命名
舉例: 一個header區塊,裡面有button元素,和樣式修飾active和disable
命名就會是：
.header
.header_button-active
.header_button-disable

picture & img

picture 本身是一個容器,沒有自己的屬性。而被包含在其中的 source 則是用於載入圖片,會包含許多圖片資訊

img 元素作為 picture 標籤中的預設圖片、當瀏覽器不支援 picture 標籤或是沒有符合的 source 時，則直接顯示 img
